RaveLink Bridge: Hue + WiZ Music Sync for Streamers

RaveLink Bridge is a local Node.js engine that turns live audio into real-time smart lighting effects for Philips Hue and WiZ. It is built for creators who want responsive music-sync visuals without cloud lag.

How It Works

The engine continuously analyzes audio level, peaks, transients, and spectrum movement, then emits scheduler-guarded lighting intents. Hue updates are rate-controlled for stability; WiZ updates stay quick for punchy response.

Quick Start

Install and run locally:

npm install
npm start

Open http://127.0.0.1:5050, pair Hue, then add WiZ IPs in Fixtures.

Who It Is For

Streamers, DJs, creators, and home setup builders looking for a local Hue/WiZ bridge with fast audio-reactive behavior and optional Twitch automation.

Extensible Mods

Includes a local mod loader with lifecycle hooks and mod HTTP endpoints so you can extend behavior without forking core server logic.

Common Questions

Can this sync Philips Hue and WiZ lights with music?
Yes. RaveLink Bridge controls both ecosystems and maps live audio behavior to light output.
Is this local or cloud-based?
Local. It runs on your machine and serves the control UI at 127.0.0.1:5050 by default.
Can Twitch channel points trigger effects?
Yes, via optional integrations and local endpoints.
Can I extend behavior with custom mods?
Yes. A mod loader is included with an example mod and reload endpoints.
